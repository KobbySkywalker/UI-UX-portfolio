Portfolio Details

ArenaGo is a user-centered UI/UX project that resulted in a mobile app tailored for sports arena events, such as football matches, concerts, and shows. The app streamlines ticketing, offers real-time event information, and fosters user engagement. It has been well-received, contributing to increased event attendance and overall customer satisfaction. Key features include intuitive navigation, digital ticketing, social engagement, and personalized event recommendations, making ArenaGo a must-have for event enthusiasts. -
